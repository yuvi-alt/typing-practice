## Getting Started

1. Clone or download this repository
2. (Optional) Add your own background music:
   - Place an audio file named `lofi.mp3` in the `assets/` folder
   - Supported formats: MP3, WAV, OGG
   - The music will start automatically when you begin typing
3. Open `index.html` in your web browser
4. Start typing!

**Note:** The audio file is not included in this repository due to size limitations. You can add your own background music or use the app without it.
